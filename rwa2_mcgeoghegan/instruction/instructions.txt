This file provides the instructions to run the rwa2_mcgeoghegan package for the RWA 2 assignment

1) Place the rwa2_mcgeoghegan in the src folder of your catkin workspace
2) In your terminal, navigate to the rwa2_mcgeoghegan package and run $ catkin build
3) Once the build command has finished, navigate back to your workspace directory and run $ source devel/setup.bash
4) If you do not have it written in your .bashrc file, run $ export TURTLEBOT3_MODEL=waffle
5) We will now launch the gazebo file for the assignment. Run $ roslaunch rwa2_mcgeoghegan rwa2.launch 
6) Once the gazebo map has loaded in open a new termianl and navigaate to your catkin workspace
7) Repeat step 4 in the new terminal
5) In your workspace run $ rosrun rwa2_mcgeoghegan my_bot_controller <x>, where <x> is the location you wish the robot to visit (0-5)
6) The waffle robot will traverse the map from its current location to the point specified in step 5
7) Once the goal has been reached the program will exit
8) Return to step 5 to have the robot travel to a new location
